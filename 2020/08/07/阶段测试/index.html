发表于 2020-08-07 更新于 2020-08-09 分类于 Linux ， Windows Sever2012 阅读次数： Valine：

阶段测试：

实验环境:

Centos 6.5 -A1 192.168.211.140
Windows server 2012 r2 -A2 192.168.211.141
Windows server 2012 r2 -A3 192.168.211.142

实验要求：

  1. 在A1上搭建VSFTPD服务，使用身份认证登录，登录用户为tom，存放文件得路径为/opt/ui

  2. 在A1上添加虚拟内存，大小为2G。设置开机自启。

  3. 在A1上创建raid5，磁盘名为r5,能正常挂载使用。

  4. 在A2上搭建域控，A3加入到域中，域名为2020kl.com,创建用户user1,user2;并且这两个用户能登录到域控制器。设置user1登录时间只能是周一到周三，并且user2不能登录到A3上；

  5. 用户登录时，不需要输入Ctrl+Alt+Del.

  6. 在A3上创建ftp，存放文件路径为c:/ftp111，并且，使用用户认证登录，只能登录到自己得家目录中，并且在各自文件夹中上传以自己名字命名得一个文件。用户名为user1,user2

解题步骤：

搭建基础环境好，基础环境，如设置IP，设置yum库，关闭防火墙等。
